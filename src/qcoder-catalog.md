## Problem Difficulty Guide

### Beginner (100-200 pts)

Perfect for starting out:

- State generation (|0⟩, |1⟩, |+⟩, |−⟩)
- Basic superposition with H gate
- Simple oracles (Copy, XOR)
- Bra-Ket notation

### Intermediate (200-400 pts)

Building proficiency:

- Multi-qubit state preparation
- Phase manipulation
- Controlled gates
- Basic arithmetic
- Shift oracles

### Advanced (400-600 pts)

For deeper understanding:

- Uniform amplitude superposition
- QFT implementation
- Modular arithmetic
- Grover's algorithm
- Zeta/Moebius transforms
- Error correction

### Expert (600+ pts)

Challenge problems:

- Hidden polynomial finding
- Complex QFT applications
- Advanced Shor's algorithm components
